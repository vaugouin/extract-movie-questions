A Python tool that parses a large PDF of general-knowledge trivia questions and uses the Google Gemini API to automatically identify and extract only the movie- and TV-related entries. For each page of the PDF, it prompts Gemini to filter questions covering topics such as films, television series, cast and crew, awards, fictional characters, plot elements, and filming locations, then writes the results to per-page CSV files with columns for page number, question number, question text, and answer. The tool supports resumable processing (skipping already-extracted pages), a configurable page range, graceful stop-on-flag behaviour, and automatic Gemini model selection, making it easy to build a focused movie-trivia dataset from a much larger question bank.
